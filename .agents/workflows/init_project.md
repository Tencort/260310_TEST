---
description: Initialize the Dolsue website project (setup, install, run dev server)
---
1. **Install project dependencies**
   ```bash
   npm install
   ```
   // turbo
2. **Run the development server**
   ```bash
   npm run dev
   ```
   // turbo
3. **Open the site in a browser**
   - Navigate to `http://localhost:3000` to view the homepage.
4. **Verify design updates**
   - Ensure all sections reflect the domestic‑assistant concept (no cleaning‑only wording).
5. **Commit initial changes**
   ```bash
   git add .
   git commit -m "Initial project setup with domestic helper branding"
   ```
   // turbo
